Kebberfegg: Generate Keyword-Based RSS Feeds

Kebberfegg generates keyword-based RSS feeds from a variety of sources. Start by entering a query that you want to monitor as a keyword-based RSS feed.

Want to learn more about how Kebberfegg works? Check out this ResearchBuzz article!


RSS Feeds Available

Tick a box and the URL for an RSS feed will appear. Beef up your feed reader! Please note that this tool is not sophisticated enough to untick the boxes once you've ticked them. When you're ready to make a new set of RSS feeds just reload the page.
